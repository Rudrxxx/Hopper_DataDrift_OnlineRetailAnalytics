## How to Produce the Final PDF Report

1. Fill in all `[Fill in]` placeholders in `project_report_template.md`
2. Add actual notebook output values (row counts, KPI numbers, t-test results)
3. Insert figure images at the appropriate sections
4. Convert to PDF using Pandoc, VS Code Markdown PDF, or copy into Google Docs → Export PDF
